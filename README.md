# Shavian language bar

This is a vibe-coded adaptation of [Adám Brudzewsky](https://github.com/abrudz) [language bars](https://github.com/abrudz/lb) to use the [Shaw JAFL keyboard layout](https://www.shavian.info/shaw_jafl_keyboard_layout/), without any installation and on pretty much any input field on any webpage.

My JavaScript knowledge is almost zero, so any comments or help are welcome.
